### 点击区域外
```js

 hidePanel (event) {
      var sp = document.querySelector(".pop");  // 选择弹出框
      if (sp) {
        if (!sp.contains(event.target)) {   // 表明点击在区域外
          this.closePop()    // 调用关闭事件
        }
      }
    }
```
